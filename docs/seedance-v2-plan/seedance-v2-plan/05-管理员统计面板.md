# M5: 管理员统计面板

## 需求描述

管理员可以查看所有成员的活动数据：谁在什么时间生成了多少视频。仅管理员可见。

## API 设计

### GET /api/admin/stats

**权限：** 仅管理员

**响应：**
```json
{
  "totalUsers": 8,
  "totalVideos": 156,
  "todayVideos": 12,
  "accountStatus": { "total": 5, "available": 3 }
}
```

### GET /api/admin/members?range=7d

**权限：** 仅管理员

**响应：**
```json
{
  "members": [
    {
      "id": 1,
      "username": "admin",
      "nickname": "ADM",
      "role": "admin",
      "videoCount": 45,
      "lastActiveAt": "2026-04-16T14:30:00Z"
    }
  ]
}
```

### GET /api/admin/recent-tasks?limit=50

**权限：** 仅管理员

**响应：** 最近生成任务列表，包含用户信息、项目信息、状态、时间。
