## 2.使用插件

重启chrome浏览器，打开插件启用

![image-20260420002636406](https://mypicture-1258720957.cos.ap-nanjing.myqcloud.com/Obsidian/image-20260420002636406.png)

 

 登录即梦平台https://jimeng.jianying.com/  （国内）打开插件

![image-20260420002753869](https://mypicture-1258720957.cos.ap-nanjing.myqcloud.com/Obsidian/image-20260420002753869.png)

打开设置seedance2.0 平台 演示平台https://seedance2.duckcloud.fun/

 设置-全局设置

![image-20260420002914160](https://mypicture-1258720957.cos.ap-nanjing.myqcloud.com/Obsidian/image-20260420002914160.png)

浏览器按 F12 打开开发者工具 → Application → Local Storage → 找到 `seedance_session_id` 的值

![image-20260420003144677](https://mypicture-1258720957.cos.ap-nanjing.myqcloud.com/Obsidian/image-20260420003144677.png)

把这个填写到seedance  chrome插件中

认证 Session ID、平台地址（Seedance Web 地址） 填写

![image-20260420003329250](https://mypicture-1258720957.cos.ap-nanjing.myqcloud.com/Obsidian/image-20260420003329250.png)

​     设置完成后点击“一键导入”  插件会自动把sessionid 填充到 seedance2.0 平台

![image-20260420003501077](https://mypicture-1258720957.cos.ap-nanjing.myqcloud.com/Obsidian/image-20260420003501077.png)

  国际版操作类似，区别在于sessionID前缀带有地区国家以及cookies

![image-20260420003837682](https://mypicture-1258720957.cos.ap-nanjing.myqcloud.com/Obsidian/image-20260420003837682.png)

 点击一键导入

![image-20260420003928080](https://mypicture-1258720957.cos.ap-nanjing.myqcloud.com/Obsidian/image-20260420003928080.png)

我们在看seedanc2.0平台

![image-20260420004021423](https://mypicture-1258720957.cos.ap-nanjing.myqcloud.com/Obsidian/image-20260420004021423.png)
